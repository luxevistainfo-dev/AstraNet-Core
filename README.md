# AstraNet-Core

Blockchain со AI слој за детекција на ризик, ботови и оптимизација.

## Како да се пушти:
1. `cd AstraNet-Core`
2. `python3 -m venv venv`
3. `source venv/bin/activate`
4. `pip install -r requirements.txt`
5. `python3 node.py`

## Команди во CLI:
- `send` - креирај трансакција
- `mine` - ископај блок
- `status` - провери состојба
- `exit` - излез
